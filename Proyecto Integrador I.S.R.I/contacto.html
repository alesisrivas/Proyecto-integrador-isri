Contacto

Este proyecto fue desarrollado para mejorar la experiencia del cliente en un restaurante mediante un sistema digital de calificación. La propuesta incluye terminales en cada mesa para que el cliente pueda valorar la atención, la comida y la comodidad, así como solicitar postre o la cuenta. Este sistema está pensado para optimizar tiempos, mejorar el servicio y recoger datos útiles para el negocio.
